# 🐶 We Care — About Your Pet

Landing page desenvolvida para uma empresa fictícia de cuidados com animais de estimação.

Este foi o meu primeiro projeto durante os estudos de desenvolvimento Front-End.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- Git e GitHub

## 📋 Funcionalidades

- Cabeçalho com menu de navegação
- Seção principal com imagem e texto
- Botões de Login e Explore
- Animações nos botões ao passar o mouse
- Utilização da fonte Montserrat
- Estilização personalizada com CSS

## 📁 Estrutura do projeto

```text
we-care-pet/
├── index.html
├── styles.css
├── amigo pet.png
└── README.md
```

## ▶️ Como executar

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto no VS Code.
3. Abra o arquivo `index.html` no navegador ou utilize a extensão Live Server.
